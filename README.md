Carlos Callagua – Filmoteca: Interfaces universales y fragmentos

Práctica de Android Básico: uso de fragments y layouts adaptativos.

* Contenido

- FilmFragmentsActivity: actividad principal para la práctica de fragments.
- FilmListFragment: muestra la lista de películas (ListFragment).
- FilmDataFragment: muestra el detalle de la película seleccionada.
- activity_film_fragments.xml: layout para pantallas pequeñas (un solo contenedor).
- layout-sw600dp/activity_film_fragments.xml: layout para pantallas grandes (lista + detalle en paralelo).

* Comprobación

- En móviles: se ve primero la lista y al pulsar una película se abre el detalle a pantalla completa.
- En tablets / pantallas grandes: se ve la lista a la izquierda y el detalle a la derecha.
